# Python Programming Paradigm Project

## Project Overview

This project includes various tasks to illustrate the core concepts of Object-Oriented Programming (OOP), exception handling, and unit testing in Python.

### Task 1: Simple Bank Account Class
- `bank_account.py`: Contains the `BankAccount` class.
- `main-0.py`: Command line interface to interact with `BankAccount`.

### Task 2: Robust Division Calculator
- `robust_division_calculator.py`: Contains the `safe_divide` function.
- `main-1.py`: Command line interface to perform division operations with error handling.

### Task 3: Unit Testing a Simple Calculator Class
- `simple_calculator.py`: Contains the `SimpleCalculator` class.
- `test_simple_calculator.py`: Unit tests for `SimpleCalculator`.

### Task 4: Library Management System
- `library_management.py`: Contains the `Book` and `Library` classes.
- `main.py`: Demonstrates interactions with the `Library` and `Book` classes.
